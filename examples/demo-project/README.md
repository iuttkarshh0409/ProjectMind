# Demo Project: TaskManager API

This is a purpose-built demo project to showcase ProjectMind's Change Impact & Review Obligation engine.

## Setup Instructions

1.  Copy the environment template:
    ```bash
    cp .env.example .env
    ```
2.  The default web server runs on port `8080`.
3.  Ensure database configuration points to your postgres instance:
    `DATABASE_URL=postgresql://localhost:5432/taskdb`
